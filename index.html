Hiernaast heb ik een display: inline-grid;
Dit houd in dat je een grid op inline ninveau maakt en kan gebruiken.

In de inline-grid heb ik een grid-template-areas gedefineerd en de span's de respectieve grid-area gegeven.

grid-template-areas: "r1 r1 r1r" "r2 r2 r2" "r3 r3 r3";

Een tip om grid te testen is om een border te geven.
border: blue 1px dashed;

Je kan zoveel articles toevoegen als je wilt en grid voegt een nieuwe row toe.
